Place work screenshots in these folders, one project per directory.

Suggested filenames:
- `cover.webp` for the grid thumbnail / primary preview
- `screen-1.webp`, `screen-2.webp`, `screen-3.webp` for additional modal or case study images

Example path:
- `/work/elyana/cover.webp`
- `/work/elyana/screen-1.webp`

Current project folders map to `src/data/projects.js` IDs.
